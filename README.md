# 🏠 meraGhar – Full-Stack PHP Project

meraGhar is a **full-stack PHP web application** designed for managing homes and properties online. It uses **PHP** for backend logic, **MySQL** for database management, and **HTML, CSS, JavaScript, Bootstrap** for frontend. An **admin panel** is included for managing users and property listings.

---

## 🚀 Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap 5  
- Backend: PHP  
- Database: MySQL  
- Authentication: PHP Sessions  
- Server: Apache / XAMPP / WAMP  

---

## 📌 Features
✅ User Registration & Login  
✅ Property Listings with Search & Filter  
✅ Add, Edit & Delete Property (Admin Panel)  
✅ Responsive UI design for mobile & desktop  
✅ Contact & Inquiry forms  
✅ Secure session management
